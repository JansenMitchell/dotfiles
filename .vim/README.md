# vim-config
My personal configuration for VIM version 8.0.
## Features
* Syntax highlighting
* Line numbers
* Search result highlighting
* Four column tab stop
* Auto indent
* Bracketed paste mode fix
* Spellcheck
  * Default for:
    * markdown
    * fountain
* Added functionality for word completion
* Text wrapping at 80 characters for:
  * markdown
  * fountain
* Linting with ALE
